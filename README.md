# ClosedClaw.com Landing Page

The official marketing site for **ClosedClaw**: The Safe Harbor for professional engineering.

## Architecture
This is a "High-Trust Industrial" static site built with:
- **Vanilla HTML5/CSS3**: For maximum performance and zero dependencies.
- **No Build Step**: "Edit and Push" simplicity.

## Deployment (GitHub Pages)

This repository is designed to be hosted directly on **GitHub Pages**.

1. Go to Settings > Pages.
2. Select Source: `Deploy from a branch`.
3. Branch: `main` / Folder: `/ (root)`.
4. Custom Domain: `closed-claw.com` (Ensure CNAME file exists).

## Visual Assets
All 3D assets are in `/assets`. They were generated using the "Nano Banana" prompt strategy for the "Sovereign Vessel" aesthetic.

## Local Development
Just open `index.html` in your browser. No `npm install` required.
